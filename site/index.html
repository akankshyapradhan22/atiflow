AtiFLOW v2.0 – MTS Tablet¶

AtiFLOW v2.0 is an industrial tablet application for the Material Transport System (MTS). It is used by floor operators (Requesters) at fixed workstations to order materials, request containers, schedule trolley returns, monitor request status, visualise staging areas, and view live inventory — all driven by Autonomous Mobile Robots (AMRs).

Target Device¶

Property Value
Screen 1024 × 640 px landscape
Touch target minimum 44 px (WCAG 2.1 AA)
Orientation Landscape only
Input method Touch (no hover dependency)
Network Always-connected (MTS-PROD)

The layout is not a responsive breakpoint design — it is fixed for the single target screen. All page dimensions are calculated around a 52 px top bar and 60 px bottom navigation, leaving 528 px of usable height inside the scrollable area.

Tech Stack¶

Layer Technology
Framework React 18 + TypeScript
Build tool Vite 5
UI library Material UI (MUI) v5
Routing React Router v6 (BrowserRouter)
State React Context API (AppContext)
Fonts Inter (Google Fonts)
Icons Material Icons (@mui/icons-material)
